# React Todo App

## 🚀 Live Demo

[Link to Live Demo](https://dmytro-halieba.github.io/react_todo-app/)

## ✨ Features

- **Full CRUD Operations**: Implemented Add, Delete, Toggle, and Rename functionality synced with a remote server.
- **Bulk Updates (Toggle All)**: Smart logic to toggle all todo statuses simultaneously with conditional styling.
- **Inline Editing**: Double-click to edit titles with support for `Enter` to save, `Esc` to cancel, and `onBlur` auto-save logic.
- **Robust Error Handling**: Integrated notification system for API failures (e.g., "Unable to update/delete a todo").
- **UX Enhancements**: Loader overlays during async operations to prevent UI flickering and improve feedback.
- **Conditional Rendering**: Dynamic "active" classes and empty-title-deletion logic.

## 🛠 Technical Highlights

- **Advanced State Management**: Utilized React Hooks (`useState`, `useEffect`, `useMemo`) for efficient state handling and side effects.
- **Clean Code**: Strict formatting and code quality enforced by Prettier and ESLint.
- **Functional Components**: Modern React architecture utilizing modular, reusable sub-components.

## 💻 Tech Stack

- **Framework**: React
- **Styling**: CSS (BEM)
- **Data Integration**: REST API
- **Language**: JavaScript (ES6+)
- **Code Quality Tools**: Prettier, ESLint

## ⚙️ Installation & Usage

To run this project locally, follow these steps:

1. **Clone the repository:**
   ```bash
   git clone <your-repository-url>
   ```
2. **Navigate to the project directory:**
   ```bash
   cd react_todo-app
   ```
3. **Install the required dependencies:**
   ```bash
   npm install
   ```
4. **Start the development server:**
   ```bash
   npm start
   ```
